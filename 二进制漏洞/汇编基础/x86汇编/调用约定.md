# 调用约定

## 1.什么是调用约定

- 实现层面(底层)的规范
- 约定了函数之间如何传递参数
- 约定了函数如何传递返回值

## 2.常见x86调用约定

- 调用者负责清理栈上的参数(Caller Clean-up)
  - cdecl
  - optlink

- 被调者负责清理栈上的参数(Callee Clean-up)
  - stdcall
  - fastcall

### 2.1 调用约定 cdecl(x86,32位)

```cpp
int callee(int a, int b, int c) {
    return a + b + c;
}

int caller(void) {
    int ret;
    ret = callee(1, 2, 3);
    ret += 4;
    return ret;
}
```
